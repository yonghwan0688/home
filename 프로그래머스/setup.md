# 프로그래머스 문제 관리 가이드

## 폴더 구조
```
프로그래머스/
├── 0단계 (입문)/
├── 1단계 (기초)/
├── 2단계 (중급)/
├── 3단계 (고급)/
└── 4단계 (최고급)/
```

## 문제 추가 방법
1. 해당 단계 폴더에 문제번호와 제목으로 폴더 생성
2. 각 폴더에 다음 파일들 생성:
   - `README.md` - 문제 설명
   - `solution.js` - 해답 코드
   - `test.js` - 테스트 케이스 (선택사항)

## 예시
```
120802. 두 수의 합 구하기/
├── README.md
├── solution.js
└── test.js
```

## Git 커밋 컨벤션
- `feat: [프로그래머스] 120802번 두 수의 합 구하기`
- `fix: [프로그래머스] 120802번 오류 수정`
